# Understanding Flexbox

Flexbox is a one-dimensional layout model that makes it easy to align and distribute space among items.

## Basic Setup

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 1rem;
}
```

## Common Patterns

### Centering an element

```css
.center {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}
```

### Space between items

```css
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
```
