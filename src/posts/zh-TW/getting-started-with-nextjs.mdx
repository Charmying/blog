# Next.js 入門指南

Next.js 是一個 React 框架，為正式環境的應用程式提供架構、功能與最佳化。

## 檔案路由系統

Next.js 使用檔案系統進行路由。只需在 `app` 目錄中建立檔案即可。

```
app/
  page.tsx        → /
  about/page.tsx  → /about
  blog/[slug]/page.tsx → /blog/:slug
```

## Server Components

在 Next.js 中，所有元件預設都是 Server Components，它們在伺服器端渲染後將 HTML 傳送給客戶端。

```tsx
export default async function Page() {
  const data = await fetch('https://api.example.com/posts');
  const posts = await data.json();

  return (
    <ul>
      {posts.map(post => (
        <li key={post.id}>{post.title}</li>
      ))}
    </ul>
  );
}
```
