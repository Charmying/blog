# React 基礎入門

React 是由 Meta 開發的前端 JavaScript 函式庫，用於建構使用者介面。

## 什麼是 JSX

JSX 是 JavaScript 的語法擴展，讓你可以在 JavaScript 中撰寫類似 HTML 的結構。

```jsx
function Hello() {
  return <h1>Hello, World!</h1>;
}
```

## 元件 (Component)

React 應用程式由元件組成。元件是可重用的 UI 區塊。

```jsx
function Card({ title, children }) {
  return (
    <div className="card">
      <h2>{title}</h2>
      {children}
    </div>
  );
}
```

## 狀態 (State)

`useState` 是 React 最基本的 Hook，用來管理元件的狀態。

```jsx
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <button onClick={() => setCount(count + 1)}>
      點擊次數：{count}
    </button>
  );
}
```
