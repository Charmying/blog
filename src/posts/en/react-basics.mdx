# React Basics

React is a frontend JavaScript library developed by Meta for building user interfaces.

## What is JSX

JSX is a syntax extension for JavaScript that lets you write HTML-like structures inside JavaScript.

```jsx
function Hello() {
  return <h1>Hello, World!</h1>;
}
```

## Components

React applications are made of components — reusable, self-contained pieces of UI.

```jsx
function Card({ title, children }) {
  return (
    <div className="card">
      <h2>{title}</h2>
      {children}
    </div>
  );
}
```

## State

`useState` is the most fundamental React Hook for managing component state.

```jsx
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <button onClick={() => setCount(count + 1)}>
      Clicked {count} times
    </button>
  );
}
```
