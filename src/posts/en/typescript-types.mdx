# TypeScript Types 101

TypeScript adds static type checking to JavaScript, helping you catch errors at compile time.

## Basic Types

```typescript
const name: string = 'Charmy';
const age: number = 25;
const isActive: boolean = true;
```

## Interface

Interfaces define the shape of an object.

```typescript
interface User {
  name: string;
  age: number;
  email?: string;
}

function greet(user: User) {
  return `Hello, ${user.name}!`;
}
```

## Generics

Generics let you write reusable, type-safe functions.

```typescript
function first<T>(arr: T[]): T | undefined {
  return arr[0];
}
```
