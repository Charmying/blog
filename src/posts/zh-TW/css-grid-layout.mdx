# CSS Grid 排版技巧

CSS Grid 是一個強大的二維佈局系統，適合用來建立複雜的頁面結構。

## 基本用法

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 1rem;
}
```

## 響應式設計

搭配 `auto-fill` 和 `minmax` 可以輕鬆實現響應式網格。

```css
.container {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
  gap: 1.5rem;
}
```
