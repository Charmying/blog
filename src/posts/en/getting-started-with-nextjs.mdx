# Getting Started with Next.js

Next.js is a React framework that provides structure, features, and optimizations for production applications.

## File-based Routing

Next.js uses the file system for routing. Simply create files inside the `app` directory.

```
app/
  page.tsx        → /
  about/page.tsx  → /about
  blog/[slug]/page.tsx → /blog/:slug
```

## Server Components

By default, all components in Next.js are Server Components. They render on the server and send HTML to the client.

```tsx
export default async function Page() {
  const data = await fetch('https://api.example.com/posts');
  const posts = await data.json();

  return (
    <ul>
      {posts.map(post => (
        <li key={post.id}>{post.title}</li>
      ))}
    </ul>
  );
}
```
